# Telegram-бот заданий за подписку

Бот: пользователь подписывается на каналы → бот автоматически проверяет подписку → начисляет ⭐ звёзды на баланс.

## Возможности

- `/start` — регистрация, меню: 👤 Профиль / ⭐ Баланс / 📋 Задание / 💸 Вывод
- **Профиль** — дата регистрации, баланс звёзд, количество выполненных заданий
- **Задание** — список каналов для подписки + кнопка «Проверить подписку» (авто-проверка через Telegram API, +20⭐ за каждый канал)
- **Вывод** — если баланс < 200⭐: «вывод доступен от 200⭐»; если ≥ 200⭐: «вывод временно недоступен»
- **Админка** (`/admin`, только для ID из ADMIN_IDS) — добавление новых каналов-заданий, просмотр и удаление каналов

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Файл `.env` уже содержит токен бота. Обязательно укажите свой Telegram ID администратора:

```
BOT_TOKEN=8049829262:AAHk8FU77_Sazh4ALFdvWufjg9jYuNIfvqc
ADMIN_IDS=ВАШ_TELEGRAM_ID
DB_PATH=bot.db
```

Свой Telegram ID можно узнать у бота **@userinfobot**. Несколько админов указываются через запятую: `ADMIN_IDS=111,222`.

⚠️ Поскольку токен был отправлен в этом чате открытым текстом, рекомендую при возможности перевыпустить его через **@BotFather** → `/revoke` (или `/token`) после настройки, чтобы исключить риск компрометации.

## Важно: бот должен быть админом в каналах-заданиях

Чтобы бот мог проверять, подписан ли пользователь на приватный канал (по ссылке-приглашению), **бот обязательно должен быть добавлен в этот канал как администратор**. Без этого `get_chat_member` вернёт ошибку, и проверка подписки для этого канала всегда будет завершаться неудачей.

## Запуск

```bash
python main.py
```

Бот будет работать в режиме long polling. Для постоянной работы на сервере используйте `systemd`, `screen`, `tmux` или `pm2`/`supervisor`.

## Админ-панель

- `/admin` — открыть панель (доступно только ID из `ADMIN_IDS`)
- **➕ Добавить канал** — бот спросит по шагам: chat_id канала → ссылку-приглашение → название → награду в ⭐ (по умолчанию 20)
- **📋 Список заданий** — показывает все активные каналы-задания с кнопкой удаления у каждого

### Как узнать chat_id канала

1. Добавьте бота в канал как администратора.
2. Перешлите любое сообщение из канала боту **@getidsbot** (или **@userinfobot**) — он покажет chat_id вида `-100xxxxxxxxxx`.

## Логика звёзд и вывода

- Баланс по умолчанию — 0
- За каждую подтверждённую подписку на канал — +20⭐ (можно изменить при добавлении канала)
- Кнопка «Вывод»:
  - баланс < 200⭐ → «Вывод доступен от 200 ⭐ на балансе»
  - баланс ≥ 200⭐ → «Вывод временно недоступен» (реального вывода в боте не реализовано — это просто информационное сообщение, как было указано в ТЗ)
